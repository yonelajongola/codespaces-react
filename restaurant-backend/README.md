# Restaurant Backend

## Setup

- Copy `.env.example` to `.env` and fill in values.
- Install dependencies: `npm install`
- Start dev server: `npm run dev`

## Routes

- `POST /auth/register`
- `POST /auth/login`
- `GET /orders`
- `PUT /orders/:id/status`
- `GET /dashboard/revenue-today`
- `GET /dashboard/orders-today`
- `GET /dashboard/revenue-7days`
- `GET /dashboard/top-items`
- `GET /dashboard/staff-performance`
- `GET /dashboard/inventory-summary`
- `GET /dashboard/ai-summary`
- `GET /dashboard/forecast`
- `GET /inventory/low-stock`
- `GET /inventory/value`
